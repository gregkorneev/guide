## **Архивирование файлов в Alt Linux**

---
## **🎯 Цель работы**

Освоить создание и распаковку архивов с помощью утилиты tar, а также научиться использовать сжатие.

---
## **🔹 1. Подготовка файлов**

  Создадим каталог и файлы для архивирования:

```
mkdir archive_test
cd archive_test

touch file1.txt file2.txt file3.txt
```

Проверка:

```
ls
```

---

## **🔹 2. Создание архива (.tar)**

```
tar -cvf archive.tar file1.txt file2.txt file3.txt
```

📌 Где:

- c — создать архив
    
- v — показать процесс
    
- f — имя файла архива
    

  

Проверка:

```
ls
```

---

## **🔹 3. Просмотр содержимого архива**

```
tar -tvf archive.tar
```

---

## **🔹 4. Извлечение архива**

```
mkdir extract
tar -xvf archive.tar -C extract/
```

Проверка:

```
ls extract
```

---

## **🔹 5. Создание сжатого архива (.tar.gz)**

```
tar -czvf archive.tar.gz file1.txt file2.txt file3.txt
```

📌 z — сжатие gzip

---

## **🔹 6. Распаковка .tar.gz**

```
mkdir extract_gz
tar -xzvf archive.tar.gz -C extract_gz/
```

---

## **🔹 7. Создание архива каталога**

```
cd ..
tar -cvf full_archive.tar archive_test/
```
