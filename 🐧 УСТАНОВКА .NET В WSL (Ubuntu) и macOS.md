## **📦 1. Установка .NET SDK (через apt — ПРАВИЛЬНЫЙ способ)**

  

### **Добавляем репозиторий Microsoft**

```
sudo apt update
sudo apt install -y wget gnupg apt-transport-https

wget https://packages.microsoft.com/config/ubuntu/$(lsb_release -rs)/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb
```

---

### **Устанавливаем SDK**

```
sudo apt update
sudo apt install -y dotnet-sdk-8.0
```

---

## **✅ Проверка**

```
dotnet --version
```

# **🍎 УСТАНОВКА .NET НА macOS**


## **📦 1. Установка через Homebrew (лучший способ)**  
### **Если нет brew:**

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

---
### **Установка .NET**

```
brew install dotnet-sdk
```

---

## **⚠️ Важно (Apple Silicon M1/M2/M3)**

Добавь в PATH:

```
echo 'export PATH="/opt/homebrew/opt/dotnet/libexec:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

---
## **✅ Проверка**

```
dotnet --version
```
