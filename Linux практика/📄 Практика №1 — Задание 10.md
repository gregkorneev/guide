## **🔹 1. Подготовка директории**

Создадим тестовую директорию с файлами:

```
mkdir mydir
cd mydir

touch file1.txt file2.txt file3.txt
cd ..
```

Проверка:

```
ls
```

---

## **🔹 2. Архивирование директории**


Создание архива:

```
tar -cvf mydir.tar mydir/
```

📌 Где:

- c — создать архив
    
- v — показать процесс
    
- f — имя архива
    

Проверка:

```
ls
```

---
## **🔹 3. Просмотр содержимого архива**

```
tar -tvf mydir.tar
```

---
## **🔹 4. Удаление исходной директории (для проверки)**

```
rm -r mydir
```

Проверка:

```
ls
```

---
## **🔹 5. Распаковка архива**

```
tar -xvf mydir.tar
```

📌 Где:

- x — извлечь файлы
    

Проверка:

```
ls
```

---
## **🔹 6. Архивирование с сжатием (.tar.gz)**

```
tar -czvf mydir.tar.gz mydir/
```

---
## **🔹 7. Распаковка сжатого архива**

```
tar -xzvf mydir.tar.gz
```

# **⚠️ Возможные ошибки**

|**Ошибка**|**Причина**|**Решение**|
|---|---|---|
|No such file|Нет директории|Проверить имя|
|Cannot open|Архив не найден|Проверить путь|
|Permission denied|Нет прав|Использовать sudo|
