## **Разметка диска и монтирование разделов в Alt Linux**

---

## **⚠️ ВАЖНО**

Работа выполняется **только с дополнительным диском** (например /dev/sdb),

НЕ трогать системный диск (/dev/sda).

---

# **🔹 1. Просмотр дисков**

```
lsblk
```

📌 Пример вывода:

```
sda    20G
├─sda1
sdb    10G
```

👉 sdb — дополнительный диск

---

# **🔹 2. Разметка диска (fdisk)**

```
su -
fdisk /dev/sdb
```

---

## **Внутри fdisk:**

### **1️⃣ Создать первый раздел**

```
n
p
1
Enter
+5G
```

---
### **2️⃣ Создать второй раздел**

```
n
p
2
Enter
Enter
```

---
### **3️⃣ Сохранить изменения**

```
w
```

---
## **🔹 Обновить таблицу разделов**

```
partprobe
```

---
## **🔹 Проверка**

```
lsblk
```

Ожидаем:

```
sdb
├─sdb1
└─sdb2
```

---
# **🔹 3. Форматирование разделов**

### **Первый раздел (ext4):**

```
mkfs.ext4 /dev/sdb1
```

---
### **Второй раздел (например xfs или ext4):**

```
mkfs.ext4 /dev/sdb2
```

(или)

```
mkfs.xfs /dev/sdb2
```

---
# **🔹 4. Создание точек монтирования**

```
mkdir /mnt/disk1
mkdir /mnt/disk2
```

---
# **🔹 5. Монтирование разделов**

```
mount /dev/sdb1 /mnt/disk1
mount /dev/sdb2 /mnt/disk2
```

---

## **🔹 Проверка монтирования**

```
df -h
```

Ожидаем:

```
/dev/sdb1  → /mnt/disk1
/dev/sdb2  → /mnt/disk2
```

---
# **🔹 6. Проверка записи**

```
touch /mnt/disk1/test1.txt
touch /mnt/disk2/test2.txt
```

```
ls /mnt/disk1
ls /mnt/disk2
```

---
# **🍎 macOS**

Все действия выполняются **через SSH**:

```
ssh username@IP
```

Далее команды выполняются внутри Alt Linux.

# **⚠️ Возможные ошибки**

|**Ошибка**|**Причина**|**Решение**|
|---|---|---|
|Device busy|Диск используется|Размонтировать|
|No such file|Неверный диск|Проверить lsblk|
|Permission denied|Нет прав|su -|
|mount failed|Нет ФС|Выполнить mkfs|
